# Results component

The results component delivers your search result via Angular Material Cards in the list or the grid view.

## Default

You need to add a results array with the specified type of `ResultEntry`.
An `<em>`-HTML tag is needed to achieve highlighting of results. Whatever is inside an `<em>` tag will be highlighted.

## Results with the grid view
